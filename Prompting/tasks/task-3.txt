Game of 24.
--------------
Rules:
You will be given a set of numbers, and you can use the basic arithmetic operations in any order or on any subset of these numbers. Your goal is to reach the number 24, or state if it is not possible.

Examples:
[3,4,5,5] : (5x5) - (4-3) = 24
[2,2,4,5] : (4x5) + 2 + 2 = 24

Solve this problem with LLM + Prompting.
